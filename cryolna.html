Project Summary

This project is currently in progress. My goal is to reproduce and understand a cryogenic CMOS LNA (sub-1-dB noise figure class) by rebuilding the key equations, validating them with Python, and then moving into Cadence/SpectreRF for full circuit-level reproduction.

Work completed so far

  • Derived and implemented the effective transconductance expression (including poles/roll-off behavior).
  • Studied the transformer-coupled tank: mode splitting, resonant frequencies, and load impedance behavior.
  • Implemented input matching analysis using Zin/S11 relationships and explored which parameters move the dip and depth.
  • Started midband gain derivations for later stages and checked how the long form collapses to standard textbook forms under assumptions.

Next Steps

  • Build the full schematic in Cadence/SpectreRF and reproduce S-parameters (S11, S21, etc.).
  • Reproduce noise figure vs. frequency and compare against the paper’s reported trends.
  • Check stability (K/μ), bias sensitivity, and gain flatness across the target band.
  • Iterate on passive Q / transformer parameters and quantify deltas (schematic vs. post-layout later).
