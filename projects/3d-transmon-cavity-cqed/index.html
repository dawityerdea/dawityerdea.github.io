Project overview

This is a two-part workflow for modeling a transmon-style Josephson element and its electromagnetic environment using Qiskit Metal and HFSS. Project 1 establishes the baseline setup (transmon + resonator/cavity workflow) and validates eigenmode simulation and field visualization. Project 2 builds directly on Project 1 by moving the sapphire and Josephson-junction location inside the cavity, then comparing how resonant frequencies, E-field localization, and Q change with geometry.

Key results (from Project 2)

After moving the sapphire and Josephson junction location, the two dominant eigenmodes show distinct behavior: Mode 1 appears high-Q (more “storage/qubit-like”), while Mode 2 is much lower-Q (more “lossy/readout-like”), consistent with the EigenQ results in the Project 2 report.

Metric Mode 1 Mode 2
Resonant frequency (example pass) ~6.2977 GHz ~7.3358 GHz
Q factor (example pass) ~8.4×107 ~9.84×103
Mode spacing Δf ≈ 1.038 GHz (reference spacing between the two dominant modes)
