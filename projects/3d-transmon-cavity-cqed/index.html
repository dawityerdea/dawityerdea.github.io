Project overview

This is a two-part workflow aimed at modeling a transmon-style Josephson element and its electromagnetic environment. Project 1 establishes a baseline setup (transmon + cavity/resonator workflow) and documents the design flow. Project 2 builds on Project 1 by moving the sapphire and Josephson-junction location inside the cavity and comparing how geometry changes affect the structure and results.
