3D Transmon and Cavity cQED (HFSS / Qiskit Metal)

HFSS eigenmode analysis • field visualization • Q factor extraction • geometry sensitivity (sapphire + JJ placement)

Project overview

This work documents a two-part workflow for modeling a 3D cavity + transmon-style Josephson element. Project 1 focuses on building the baseline cavity model and validating the eigenmode setup. Project 2 builds directly on Project 1 by changing the sapphire and Josephson-junction placement, then comparing how mode frequencies, field localization, and Q change with geometry.

What I did (portfolio highlights)
  • Set up HFSS eigenmode simulation (multiple modes, convergence tuning, pass control).
  • Compared eigenmode frequencies between baseline geometry (Project 1) and modified geometry (Project 2).
  • Used E-field plots (top + side views) to interpret where energy concentrates (cavity vs. qubit region).
  • Extracted Q from EigenQ results and interpreted “high-Q storage-like” vs “lossy/readout-like” behavior.
  • Documented practical troubleshooting (convergence sensitivity to max passes) and next steps for nonlinear analysis.
Key results (from Project 2)

After moving the sapphire and Josephson junction location, the two main modes show distinct behavior: Mode 1 appears high-Q (storage/qubit-like), while Mode 2 is much lower-Q (more lossy/readout-like).

Metric Mode 1 Mode 2
Resonant frequency (example pass) ~6.2977 GHz ~7.3358 GHz
Q factor (example pass) ~8.4×107 ~9.84×103
Mode spacing Δf ≈ 1.038 GHz (used as a qualitative indicator for weaker/stronger coupling)

Note: extracting a numeric cross-Kerr (χ) generally requires nonlinear modeling (e.g., pyEPR / Josephson nonlinearity), so the current results emphasize eigenmodes, field overlap intuition, and Q/frequency sensitivity to geometry.
