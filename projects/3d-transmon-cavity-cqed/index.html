3D Transmon & Cavity cQED (Qiskit Metal + HFSS)

Qiskit Metal layout • HFSS 3D simulation • results documented in two short reports

In progress Qiskit Metal HFSS
Project overview

This work is split into two small reports. Both use the same general workflow: build a transmon/cavity design in Qiskit Metal, then run 3D simulations in HFSS to study how the design behaves.

  • Project 1 (PDF 1): Baseline setup. Shows the initial design, how it is set up in HFSS, and the first simulation results.
  • Project 2 (PDF 2): Design change study. Moves the sapphire / junction location and compares how the results change versus Project 1.
View Project 1 (baseline) PDF → View Project 2 (comparison) PDF →
What I did
  • Built the transmon + cavity layout in Qiskit Metal using adjustable settings.
  • Imported the design into HFSS and ran eigenmode simulations.
  • Reviewed the simulation outputs (frequencies, field plots, and Q factor) and summarized them in the two PDFs.
Status / next steps
