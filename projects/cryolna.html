Cryo Sub-1-dB NF CMOS LNA (CryoLNA)

Cadence/SpectreRF • Gain / NF / stability • reproduction study

Status: Under development

This project is in progress. I’m reproducing a cryogenic CMOS LNA by rebuilding key derivations, validating them with analysis scripts, and then reproducing circuit-level results in Cadence/SpectreRF. The full code and internal workbench are kept private while development is ongoing.

Work completed so far
  • Derived effective transconductance expressions and checked frequency behavior.
  • Analyzed the coupled/transformer tank (resonances, splitting behavior).
  • Started input match analysis (Zin/S11) and parameter sensitivity.
Next steps
  • Reproduce S-parameters and NF in SpectreRF and compare against the reference.
  • Stability checks (K/μ), bias sweeps, and sensitivity study.
  • Track schematic vs. post-layout deltas later.
