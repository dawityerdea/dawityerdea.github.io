Project Summary

During my SLAC FACET-II internship, I worked on orbit fitting using BPM measurements and a linear transport model. The workflow reconstructs the upstream orbit using least-squares fitting, then uses residual position at spectrometer BPMs to estimate relative energy deviation.

  • Input: BPM position data across multiple beamline sections.
  • Model: linear transport using an R-matrix map between BPMs (horizontal plane).
  • Fit: solve an overdetermined system by least-squares / pseudo-inverse to estimate the initial state.
  • Diagnostics: convert spectrometer BPM residuals into δ (relative energy error).

Diagnostics Method (Energy from Residuals)

After fitting upstream BPMs, I predict the betatron-only position at a spectrometer BPM and compute the residual:

x_res = x_meas − x_fit

In a dispersive region, the residual is related to energy deviation. Using dispersion η (often from R16):

δ ≈ x_res / η

Convert to energy offset using a nominal beam energy:

ΔE = E_nominal · δ
Example spectrometer BPMs BPM10731, BPM11333, BPM14801
Sections used DL10, BC11, BC14 (multi-point energy check)
