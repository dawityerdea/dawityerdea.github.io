Project Summary

This project demonstrates an end-to-end RF/microwave workflow: specification-driven design, ADS schematic + layout, Momentum EM analysis, fabrication on a PCB milling machine, and practical build documentation (including lessons learned).

  • Design: Chebyshev band-pass filter and microwave coupler structures built and documented in ADS.
  • EM iteration: Used Momentum to validate and refine layout behavior beyond ideal transmission-line models.
  • Fabrication: Generated manufacturable artwork and executed milling/processing steps.
  • Validation: Captured measurement-style evidence (VNA screen photos) and build photos.
Filter design (PDF) Branchline coupler (PDF) EM revisions (PDF) ADS workspace (ZIP)

Design Targets

Band 2.0–3.0 GHz (passband focus: 2–3 GHz)
Passband ripple 0.3 dB (±0.1 dB)
Excess loss < 0.5 dB
Out-of-band rejection > 40 dB for < 2.2 GHz and > 2.7 GHz
Return loss (passband) > 11 dB
